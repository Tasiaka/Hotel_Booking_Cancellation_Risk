# Логин и регистрация

В Streamlit UI добавлена простая локальная регистрация для MVP.

## Демо-доступ

```text
admin / admin   администратор
user / user     менеджер
```

## Регистрация

На странице входа выберите режим **Регистрация**, задайте логин, отображаемое имя и пароль. Новый аккаунт получает роль менеджера.

Ограничения логина:

```text
3–32 символа: латиница, цифры, точка, дефис, подчеркивание
```

## Хранение пользователей

Зарегистрированные пользователи сохраняются локально:

```text
storage/users.json
```

Пароли не сохраняются открытым текстом. Для учебного MVP используется PBKDF2-SHA256 hash с солью.

## Роли

Саморегистрация не выдает роль администратора. Админский контур доступен только через demo-аккаунт:

```text
admin / admin
```

Админская вкладка **Пользователи** показывает demo-пользователей и зарегистрированные аккаунты менеджеров.
