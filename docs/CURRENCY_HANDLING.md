# Валюта денежных показателей

Сервис не выполняет автоматическую конвертацию валют. Значение `adr` берется как число из CSV, а производные поля считаются в той же валюте:

```text
booking_value = adr × total_nights
expected_loss = risk_score × booking_value × deposit_loss_factor
```

В интерфейсе менеджер выбирает валюту отображения для загруженного файла:

```text
RUB ₽
EUR €
USD $
GBP £
```

Выбор валюты меняет только знак в KPI, таблицах и бизнес-симуляторе. Если в одном CSV смешаны разные валюты, результат нельзя интерпретировать как единую сумму: такие данные нужно предварительно нормализовать вне сервиса.
