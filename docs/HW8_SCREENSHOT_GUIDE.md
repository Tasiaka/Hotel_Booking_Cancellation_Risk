# Гайд по скриншотам для ДЗ №8

В отчет `homework_08/packaging_mvp_report.md` нужно вставить скриншоты работающего MVP.

Минимальный набор:

1. Структура проекта в IDE или терминале.
2. `GET /api/v1/domain-model`.
3. Схема/таблицы СУБД или файл `storage/hotel_risk.db`.
4. Вкладка UI «История скорингов».
5. Swagger UI `/docs`.
6. Ответ `/api/v1/score-csv` или `/api/v1/batches/upload`.
7. Главный экран UI после скоринга.
8. Загрузка CSV / demo dataset.
9. Список броней к проверке.
10. Карточка брони.
11. Метрики модели или `reports/tables/hw8_model_metrics.json`.
12. Результат `pytest`.
13. Результат `docker compose up --build`.
14. Масштабирование `docker compose up --scale worker=3`.

Публичный домен для этого домашнего задания не требуется. Достаточно локального запуска и GitHub README.
